## 📋 Admin Tabs Explained

### 1. **Pages Tab**
Select which page you want to edit. See all pages at once with section count.

**What You Can Do:**
- Click any page card to select it
- See number of sections on each page
- Selected page gets amber border and background

### 2. **Sections Tab** (Most Important!)
Add, edit, reorder, and delete sections on the current page.

#### Section Management:

**Add a Section:**
1. Click "Add Section" button
2. Choose section type from dropdown:
   - Hero Carousel - Full-width hero with slides
   - Featured Products - Showcase featured items
   - Best Sellers - Top selling items
   - Categories - Product categories
   - Testimonials - Customer reviews
   - Features - Key features list
   - Text Section - Custom text content
   - Image Gallery - Image showcase
   - Contact Form - Contact section
   - FAQ - Frequently asked questions
   - Stats - Statistics display
   - Team - Team members
   - CTA Banner - Call-to-action
   - Newsletter - Newsletter signup
   - Products Grid - Product listing

3. Click "Create Section"

**Edit a Section:**
1. Click "Edit" button (pencil icon) on any section
2. Modify these properties:
   - **Section Title** - What appears at top
   - **Layout** - Choose display format:
     - Full Width - Spans entire width
     - Centered - Centered content
     - Two Column - Split into 2 columns
     - Three Column - Split into 3 columns
     - Four Column - Split into 4 columns
   - **Background Color** - Section background (pick color)
   - **Text Color** - What color text appears in
   - **Description** - Additional info
   - **Images** - Add multiple images to section

3. Click "Done Editing"

**Reorder Sections:**
- Use **Up/Down arrows** to move sections around
- Top section appears first on page

**Show/Hide Sections:**
- Click **Eye icon** to toggle visibility
- Hidden sections don't appear on website but stay saved

**Delete Sections:**
- Click **Trash icon** to remove section
- Click OK to confirm

#### Section Colors Guide:
```
- Background Color: What color the section background is
- Text Color: What color the text/content appears in

Examples:
- Dark background + white text = high contrast
- Light background + dark text = easy to read
- Amber background + white text = branded look
```

### 3. **Colors Tab**
Customize your entire site's color theme instantly.

**All Theme Colors:**
- **Primary** - Main color (usually amber #f59e0b)
- **Secondary** - Secondary accent color
- **Accent** - Dark brand color
- **Dark** - Dark backgrounds (footer, header)
- **Light** - Light backgrounds
- **Text** - Regular text color
- **Success** - Green checkmarks, success messages
- **Error** - Red error/warning messages

**How to Change:**
1. Click the color swatch square
2. Choose new color
3. OR paste hex code (e.g., #f59e0b) directly in text field
4. See instant preview on right side

**Example Themes:**
```
Professional:
- Primary: #3b82f6 (Blue)
- Accent: #1e3a8a (Dark Blue)
- Text: #1f2937 (Dark Gray)

Warm:
- Primary: #ea580c (Orange)
- Accent: #7c2d12 (Dark Orange)
- Text: #0f172a (Dark Slate)

Green/Eco:
- Primary: #16a34a (Green)
- Accent: #15803d (Dark Green)
- Text: #0f172a (Dark Slate)
```

### 4. **Settings Tab**
Configure the current page's properties.

**Settings Available:**
- **Page Title** - What appears in browser/SEO
- **Page Slug** - URL path (e.g., /about for About page)
- **Page Description** - SEO description
- **Visible** - Toggle if page shows on website

## 🎨 Complete Theme Configuration Guide

### Step-by-Step Theme Customization

1. **Go to Colors Tab** in Page Builder
2. **Primary Color** - Your brand main color
   - Used for buttons, links, highlights
   - Choose something that stands out
   - Example: #f59e0b (Amber), #3b82f6 (Blue), #10b981 (Green)

3. **Secondary Color** - Complementary color
   - Used for hover states, accents
   - Usually lighter version of primary
   - Example: #fbbf24 (Light Amber)

4. **Accent/Dark** - Dark backgrounds
   - Header, footer, dark sections
   - Should be very dark
   - Example: #0f172a (Dark Slate)

5. **Light** - Light backgrounds
   - Page backgrounds, light sections
   - Usually off-white
   - Example: #f8fafc (Light Slate)

6. **Text Colors** - What text looks like
   - Regular text: main content color
   - Success/Error: feedback messages

7. **Click Preview** - See changes instantly in right panel

8. **Auto-saves** - All color changes saved automatically

## 🔧 Troubleshooting

### Changes Not Showing?
1. Hard refresh (Ctrl+Shift+R)
2. Clear browser cache
3. Check localStorage in DevTools (F12 → Application → LocalStorage)

### Section Text Color Too Dark?
1. Go to Colors Tab
2. Click the Text Color
3. Choose lighter color (e.g., white #ffffff)

### Images Not Uploading?
1. Check file size (keep under 5MB)
2. Ensure image format (JPG, PNG, GIF)
3. Try different image

### Want to Reset Everything?
1. Open browser DevTools (F12)
2. Go to Application → LocalStorage
3. Find `wudden_site_builder_config`
4. Delete it
5. Refresh page

## 💡 Pro Tips

### Tip 1: Color Harmonies
```
Monochromatic (sophisticated):
- Use variations of same color
- Primary: #f59e0b, Secondary: #fbbf24

Complementary (bold):
- Pick opposite colors
- Primary: #f59e0b (amber), Accent: #3b82f6 (blue)

Analogous (cohesive):
- Pick neighboring colors on wheel
- Primary: #f59e0b, Secondary: #f97316
```

### Tip 2: Section Hierarchy
```
Good order:
1. Hero (attention-grabber)
2. Featured (best products)
3. Best Sellers (social proof)
4. Categories (navigation)
5. CTA (call-to-action)
6. Footer (info)
```

### Tip 3: Contrast for Readability
- Dark text on light = always readable
- Light text on dark = always readable
- Dark text on light background! = hard to read
- Make sure text color contrasts with background

### Tip 4: Not All Sections Need Images
- Text sections work great without images
- Keep loading fast with minimal images
- Use images strategically

### Tip 5: Mobile Friendly
- 4-column layout → might be 1 column on mobile
- Keep titles short
- Use readable font sizes
