# Page Builder - Quick Examples & Use Cases

## 🎯 Common Tasks

### Task 1: Change Your Brand Colors

**Goal:** Rebrand from amber to blue

**Steps:**
1. Admin Dashboard → Page Builder
2. Click **Colors Tab**
3. Under "Primary Color" → click color square
4. Choose blue: `#3b82f6`
5. Watch preview update on right
6. Done! All buttons now blue

**Time:** 1 minute

---

### Task 2: Add a New Section to Homepage

**Goal:** Add a testimonials section between Best Sellers and Categories

**Steps:**
1. Admin Dashboard → Page Builder
2. Click **Pages Tab** → select "Home"
3. Click **Sections Tab**
4. Find "Best Sellers" section
5. Click down arrow to move it down
6. Click "Add Section" button
7. Choose "Testimonials" from dropdown
8. Click "Create Section"
9. Done! Testimonials now between bestsellers and categories

**Time:** 2 minutes

---

### Task 3: Hide a Section Temporarily

**Goal:** Hide the newsletter section without deleting it

**Steps:**
1. Admin Dashboard → Page Builder
2. **Sections Tab**
3. Find "Newsletter" section
4. Click **Eye icon** to hide
5. Section still exists but won't show on website
6. Click eye again to show later

**Time:** 10 seconds

---

### Task 4: Change Section Colors

**Goal:** Make the "Featured Products" section have a blue background with white text

**Steps:**
1. Admin Dashboard → Page Builder
2. **Sections Tab** → find "Featured Products"
3. Click **Edit** (pencil icon)
4. **Background Color** → click color picker
5. Choose blue: `#3b82f6`
6. **Text Color** → click color picker
7. Choose white: `#ffffff`
8. Click "Done Editing"
9. Section now blue with white text!

**Time:** 1 minute

---

### Task 5: Reorder All Homepage Sections

**Goal:** Put categories first, then featured products, then bestsellers

**Current Order:**
1. Hero
2. Featured Products
3. Best Sellers
4. Categories

**Desired Order:**
1. Hero
2. Categories
3. Featured Products
4. Best Sellers

**Steps:**
1. Admin Dashboard → Page Builder
2. **Sections Tab** → Home page
3. Find "Featured Products" → click **Down arrow** once (moves to 3rd)
4. Find "Best Sellers" → click **Down arrow** once (moves to 4th)
5. Done! New order: Hero → Categories → Featured → Bestsellers

**Time:** 1 minute

---

### Task 6: Add Images to a Section

**Goal:** Add 3 testimonial images to the new testimonials section

**Steps:**
1. Admin Dashboard → Page Builder
2. **Sections Tab**
3. Find "Testimonials" section
4. Click **Edit** (pencil icon)
5. Scroll to **"Section Images"**
6. Click "**+ Add Image**"
7. Choose first image from computer → uploads
8. Click "**+ Add Image**" again
9. Choose second image → uploads
10. Click "**+ Add Image**" again
11. Choose third image → uploads
12. Click "Done Editing"
13. Done! 3 images now in testimonials

**Time:** 2 minutes

---

### Task 7: Create Light/Dark Sections

**Goal:** Alternate between light and dark sections for visual interest

**Example Layout:**
- Section 1: Dark background (slate #0f172a) + white text
- Section 2: Light background (white #ffffff) + dark text
- Section 3: Dark background + white text
- Section 4: Light background + dark text

**Steps:**
1. Admin Dashboard → Page Builder
2. **Sections Tab**
3. Click **Edit** on Section 1
4. Set Background Color: `#0f172a` (dark)
5. Set Text Color: `#ffffff` (white)
6. Click "Done"
7. Click **Edit** on Section 2
8. Set Background Color: `#ffffff` (white)
9. Set Text Color: `#0f172a` (dark)
10. Click "Done"
11. Repeat pattern for sections 3 & 4

**Result:** Nice alternating dark/light sections

**Time:** 3 minutes

---

### Task 8: Change Page Title & Description

**Goal:** Update Home page title and meta description for SEO

**Steps:**
1. Admin Dashboard → Page Builder
2. Click **Pages Tab** → select "Home"
3. Click **Settings Tab**
4. **Page Title:** Change to "Premium Interior Design Furniture | Wudden Interior"
5. **Page Description:** Change to "Shop our collection of premium interior design products. Free shipping on orders over ₹5000. 30-day returns."
6. Done!

**Time:** 1 minute

## 📱 Responsive Design Tips

### What Happens on Mobile?

**Desktop:**
- 4-Column layout → shows 4 columns side-by-side
- 3-Column → shows 3 columns side-by-side
- 2-Column → shows 2 columns side-by-side

**Tablet (768px):**
- 4-Column → shows 2 columns (2x2 grid)
- 3-Column → shows 2 columns (wrap)
- 2-Column → shows 1 column

**Mobile (375px):**
- All layouts → show 1 column (stacked)
- All sections stack vertically
- Images scale to width

**Pro Tip:** Test your layout on phone to ensure it looks good!

## ⚡ Performance Tips

### Optimize for Speed

1. **Fewer Large Images**
   - Use max 2-3 images per section
   - Compress images before upload
   - Target 100-200KB per image

2. **Minimize Sections**
   - Stick to 5-7 sections per page
   - Hidden sections still load
   - Remove unused sections

3. **Simple Layouts**
   - 4-column grids load fast
   - Fewer custom styles = faster
   - Stick to standard layouts

## 🆘 Quick Fixes

### Problem: Text Too Hard to Read
**Solution:** Go to Sections → Edit → Change Text Color to white or dark

### Problem: Section Looks Empty
**Solution:** Add images or check if Text Color matches Background

### Problem: Layout Looks Weird on Mobile
**Solution:** Reduce columns (try 2-column instead of 4)

### Problem: Colors Don't Match Brand
**Solution:** Go to Colors Tab → update Primary/Secondary/Accent

### Problem: Changes Not Showing
**Solution:** Hard refresh (Ctrl+Shift+R) or clear localStorage
